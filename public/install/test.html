Troubleshooting Guide:

If PHP files show "Internal Server Error":

  1. Check if PHP is enabled for this directory
  2. Check file permissions (should be 644 for PHP files)
  3. Check .htaccess rules
  4. Check server error logs

Manual Fix for .htaccess:

Create or edit .htaccess in root directory with:

RewriteEngine On

# Allow access to install directory
RewriteRule ^install/ - [L]

# Allow access to build assets  
RewriteRule ^build/ - [L]

# Laravel routing
RewriteCond %{REQUEST_FILENAME} !-d
RewriteCond %{REQUEST_FILENAME} !-f
RewriteRule ^ index.php [L]
